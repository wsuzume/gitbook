# Graph Regularized Non-negative Matrix Factorization for Data Representation

## メモ

グラフ正則化項という若干珍しい正則化項付きのNMF。各変数がグラフ上で定義されていて隣接行列が判明している（たとえば画像において画素と周辺の画素どうしの繋がりが明示的に定義されている）ような状況で使える。
